# Address Book

## Description
A console-based address book in C that uses structures to store contacts and allows adding and viewing contacts.

## Features
- Add multiple contacts
- View all saved contacts
- Uses structures for data organization

## How to Run
1. Open terminal/command prompt
2. Navigate to the Address-Book folder
3. Compile using: gcc address_book.c -o address_book
4. Run using: ./address_book

## Technologies Used
- C programming
- Structures in C
